## 项目背景

工艺管理软件是一个面向**工业制造领域**的工艺流程管理系统，旨在实现工艺过程的数字化管理和标准化控制。

### 应用场景

- **工艺卡管理**：创建、审批、执行工艺卡片
- **产品轨迹追溯**：记录产品加工全流程
- **标定数据管理**：粗糙度、长度等测量标定
- **缺陷管理**：缺陷类型、验收标准、检测标准
- **权限精细控制**：基于角色的三级权限体系

## 系统特点

### 1. 前后端完全分离

**前端**：Vue 3 单页应用（SPA），可部署到任何静态服务器  
**后端**：Spring Boot RESTful API，无状态设计

**优势**：

- 前后端独立部署和扩展
- 支持多环境配置（dev/test/prod）
- 灵活的连接配置（首次访问时配置后端地址）

### 2. 三级权限控制

**用户 → 角色 → 权限** 的完整 RBAC 体系：

- **用户（User）**：系统使用者
- **角色（Role）**：如 admin、reviewer、executor 等
- **权限（Permission）**：具体操作权限，如 "用户管理"、"工艺卡创建" 等

**特性**：

- 一个用户可拥有多个角色
- 一个角色可拥有多个权限
- 权限采用数字编码体系（如 "11" 用户管理、"41" 工艺卡创建）
- 支持细粒度权限（如工艺卡创建者仅能编辑自己的卡）

### 3. JWT 无状态认证

- **Access Token**：有效期 1 小时
- **Refresh Token**：有效期 7 天
- 自动刷新机制
- 防暴力破解（LoginAttempt 记录）

### 4. 丰富的组件库

- **Table 组件系列**：table-custom、table-edit、table-search、table-detail
- **选择器组件**：DefectSelector、RoughnessSelector、ModelSelector
- **文件上传**：支持分块上传大文件
- **权限指令**：v-permiss 自动控制元素显示

### 5. 数据库版本控制

- **开发阶段**：使用 JPA 自动创建数据库（ddl-auto: create-drop）
- **生产阶段**：使用 Flyway 管理数据库迁移
- 支持三环境：dev（开发）、test（测试）、prod（生产）

## 核心功能模块

### 系统管理

- **用户管理**：创建、编辑、删除用户，分配角色
- **角色管理**：创建角色，分配权限
- **权限管理**：创建权限，按分类组织
- **菜单管理**：配置系统菜单结构

### 标定管理

- **粗糙度标定**：管理粗糙度测量标准
- **长度标定**：管理长度测量标准
- **标定信息管理**：记录标定历史

### 缺陷管理

- **缺陷类型**：定义缺陷分类和等级
- **验收标准**：设置缺陷验收规则
- **检测标准**：设置缺陷检测规则
- **模型管理**：管理检测模型文件

### 工艺卡管理

- **工艺卡 CRUD**：创建、查看、编辑、删除
- **审批流程**：审核 → 确认 → 执行
- **权限细分**：创建、删除、修改、执行、审核、确认、导出、查看
- **文件导出**：导出 Excel 格式工艺卡

### 产品轨迹管理

- **轨迹规划**：为产品规划加工轨迹
- **轨迹执行**：记录实际加工过程
- **数据追溯**：查询产品完整轨迹

## 技术架构

### 后端技术栈

```
Java 21
├── Spring Boot 3.4.4
│   ├── Spring Security（安全框架）
│   ├── Spring Data JPA（数据持久化）
│   └── Spring Web（RESTful API）
├── JWT（jjwt 0.12.6）
├── MyBatis（SQL 映射）
├── MySQL 8.0+
├── Flyway（数据库迁移）
└── Lombok（代码生成）
```

### 前端技术栈

```
Vue 3.4.5
├── Vite 3.0.0（构建工具）
├── Pinia 2.1.7（状态管理）
├── Element Plus 2.6.3（UI 组件库）
├── Axios 1.6.3（HTTP 客户端）
├── Vue Router 4.2.5（路由）
├── TypeScript 4.6.4
└── Tauri 2.1.0（可选桌面应用）
```

## 开发团队角色

系统内置以下角色：

| 角色 | 角色标识 | 说明 |
|------|---------|------|
| **管理员** | admin | 拥有所有权限 |
| **审核员** | reviewer | 可审核工艺卡 |
| **确认员** | confirmer | 可确认工艺卡 |
| **执行员** | executor | 可执行工艺卡 |
| **普通用户** | user | 基础权限 |
| **开发人员** | developer | 所有权限（含权限管理） |
